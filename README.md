# thesea

<img width="1912" height="895" alt="Screenshot 2026-05-09 000727" src="https://github.com/user-attachments/assets/7c6518f3-94fb-4717-9f74-6e1068bc64c9" />

TheSea is a simple and fun sandbox ecosystem survival game where the user can do their part to maintain a balanced ecosystem where all participants have needs and impacts upon others, forming a dynamic set of relationships within their environment and those they share it with. 

<img width="1912" height="895" alt="Screenshot 2026-05-09 000727" src="https://github.com/user-attachments/assets/6951691b-c7de-416c-afc9-624beb6000cb" />

Or the user can sit back and watch the ecosystems' forces have its way upon the system, idly observing the growth and then decay of a complex simulated organism, which mirrors the growth and decay as well as continuous change which both strongly nod to the absurdity and difficulty of managing one's own life in the world today.
